PROJETO ALPHATEC

O Alphatec é um projeto focado na criação de um carrinho autônomo capaz de navegar de forma independente em diversos ambientes. Utilizando tecnologias como visão computacional e sensores ultrassônicos.

Projeto

Ferramentas Utilizadas

Softwares Utilizados

Onshape para modelagem do chassi

Visual Studio Code para criação do portfólio do projeto

Aplicativo Arduino para fazer a parte de movimentação autônoma do carro

Componentes Usados

- Placa Arduino UNO R3

- Chassi

- Roda + Pneu + Motor Dc 6v

- Módulo Motor Ponte H

- Módulo Sensor Ultrasônico Arduino

- Chave Liga/desliga

- Micro Servo-motor

Projeto Fundamentos Tecnológicos II

Desafios enfrentados

Tivemos muitos empecilhos, em parte por falta de conhecimento dos integrantes, principalmente em relação ao código e o planejamento

Aprendizados

Dado os empecilhos, tivemos que aprender muito durante o projeto, nos organizamos para conseguir terminar o projeto mesmo que atrasados e nos ajudamos nas partes que podíamos
